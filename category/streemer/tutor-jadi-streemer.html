Cara Mendapatkan Rank Streamer di Ghost SMP

Pendahuluan

Rank Streamer adalah status istimewa di server Ghost SMP yang diberikan kepada anggota komunitas yang aktif melakukan streaming atau konten kreasi yang terkait dengan server.

Persyaratan Umum

  • Memiliki platform streaming aktif (YouTube, Twitch, Facebook Gaming)
  • Minimal 100 subscriber atau follower
  • Telah melakukan streaming minimal 3 kali tentang server Ghost SMP
  • Memiliki rekaman atau bukti streaming yang dapat diverifikasi

Prosedur Pengajuan

  1. Persiapan Dokumen
    • Kumpulkan link channel streaming
    • Sertakan tangkapan layar statistik channel
    • Dokumentasikan minimal 3 video streaming server Ghost SMP
  2. Pengajuan
    • Kunjungi halaman "Pencalonan" di website resmi Ghost SMP
    • Isi formulir pengajuan Rank Streamer
    • Lampirkan bukti-bukti yang diperlukan
  3. Verifikasi
    • Tim staff akan meninjau pengajuan dalam waktu 3-5 hari kerja
    • Akan dilakukan pengecekan keabsahan dokumen dan channel streaming

Kriteria Penilaian

  • Kualitas konten streaming
  • Jumlah viewer rata-rata
  • Kontribusi terhadap promosi server Ghost SMP
  • Keaktifan dalam komunitas server

Manfaat Rank Streamer

  • Badge khusus di dalam server
  • Akses ke area eksklusif
  • Kemungkinan kerja sama sponsorship
  • Prioritas dalam event-event server

Catatan Tambahan

  • Rank Streamer bersifat tidak permanen dan akan ditinjau ulang setiap 3 bulan
  • Penyalahgunaan rank dapat mengakibatkan pencabutan status
